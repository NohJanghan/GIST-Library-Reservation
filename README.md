## Overview
GIST 도서관의 시설예약을 위한 정적 웹 프로젝트입니다.  
다음과 같은 기능을 제공하는 것을 목표로 하고 있습니다.  

* 모바일 친화적 UI  
* 사용 가능한 공간 자동 선택  
* PWA(Progressive Web App)
  - 예약 시간 전에 푸시 알림 
* 예약 내역 관리\
